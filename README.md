# JobsDB 自动求职工具

一个基于 Python 和 DeepSeek API 的自动化求职工具，支持自动搜索岗位、生成定制简历、自动投递等功能。

## 功能特点

1. **初始化配置**
   - API密钥配置
   - Chrome浏览器配置（支持使用已登录的Chrome配置文件）
   - 个人信息配置（姓名、邮箱、电话）

2. **全自动求职**
   - 自动搜索JobsDB岗位
   - 批量抓取岗位详情
   - AI匹配度计算
   - 自动生成定制简历
   - 自动投递（支持暂停/继续）

3. **手动单岗处理**
   - 手动输入岗位链接
   - 抓取岗位描述
   - 生成定制简历
   - 导出PDF

4. **投递记录查询**
   - 查看所有投递记录
   - 导出记录为CSV/Excel

## 安装步骤

1. **安装Python**
   - 确保已安装Python 3.7或更高版本

2. **安装依赖**
   ```bash
   pip install -r requirements.txt
   ```

3. **配置API密钥**
   - 在`api_config.json`中配置DeepSeek API Key（如果使用）
   - 或在程序界面的「初始化配置」标签页中配置

4. **运行程序**
   ```bash
   python jobsdb_ai_tool.py
   ```

## 使用说明

### 1. 初始化配置

- **API密钥配置**：输入DeepSeek API Key（必填）
- **Chrome浏览器配置**：
  - 选择Chrome用户数据目录（可选，用于使用已登录的Chrome）
  - 选择配置文件名称（默认：Default）
- **个人信息配置**：填写姓名、邮箱、电话

### 2. 全自动求职

1. 完成「初始化配置」标签页的配置（包括上传简历）
2. 在「全自动求职」标签页填写搜索条件：
   - 地区选择
   - 搜索关键词
   - 搜索地点
   - 匹配度阈值（建议70）
3. 点击「开始全自动求职」按钮
4. 程序将自动搜索、匹配、生成简历并投递

### 3. 手动单岗处理

1. 在「手动单岗处理」标签页输入岗位链接
2. 点击「抓取岗位信息」按钮
3. 选择简历语言（auto/zh/en）
4. 点击「生成定制简历」按钮
5. 可导出为PDF

### 4. 投递记录查询

- 在「投递记录查询」标签页查看所有投递记录
- 点击「导出记录」按钮可导出为CSV或Excel

## 注意事项

1. **Chrome浏览器**：需要安装Chrome浏览器，程序会自动下载ChromeDriver
2. **API密钥**：需要有效的DeepSeek API Key才能使用AI功能
3. **投递频率**：程序已内置投递频率控制（每日最多15个，间隔6-12分钟），避免被网站封禁
4. **简历语言**：生成的简历语言可以选择自动检测、中文或英文

## 文件说明

- `jobsdb_ai_tool.py`：主程序文件
- `config.json`：用户配置文件（自动生成）
- `api_config.json`：API配置文件（可选）
- `application_records.json`：投递记录文件（自动生成）
- `resume_cache.txt`：简历缓存文件（自动生成）
