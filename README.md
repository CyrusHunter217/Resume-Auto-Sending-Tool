# JobsDB 自动求职工具

一个基于 Python 和 DeepSeek API 的自动化求职工具，可以自动搜索岗位、生成定制简历、并协助投递申请。

## 功能特性

### 1. 初始化配置
- **API 密钥配置**：支持 DeepSeek API Key 配置
- **浏览器配置**：支持使用现有 Chrome 用户配置文件（保持登录状态）
- **个人信息**：配置姓名、邮箱、电话等投递信息

### 2. 全自动求职
- **地区选择**：支持香港、新加坡、马来西亚、菲律宾
- **搜索条件**：关键词、地点、匹配度阈值设置
- **自动搜索**：自动搜索匹配岗位（需根据实际网页结构定制）
- **暂停/继续**：支持暂停和继续自动求职流程

### 3. 手动单岗处理
- **岗位信息抓取**：从 JobsDB 链接自动抓取岗位描述
- **简历生成**：基于岗位描述和原始简历，使用 AI 生成定制简历
- **简历上传**：支持上传 Word 格式简历
- **简历预览**：预览原始简历和生成的定制简历
- **PDF 导出**：将生成的简历导出为 PDF 格式

### 4. 投递记录查询
- **记录管理**：自动记录所有投递的岗位信息
- **记录查询**：查看投递历史记录（岗位名称、公司、投递日期、匹配度、状态）
- **记录导出**：支持导出为 CSV 或 Excel 格式

### 5. 其他功能
- **中英文切换**：支持界面语言切换
- **配置自动保存**：配置信息自动保存到本地
- **匹配度计算**：自动计算简历与岗位的匹配度

## 安装说明

### 前置要求
- Python 3.8 或更高版本
- Chrome 浏览器
- DeepSeek API Key（从 [DeepSeek](https://platform.deepseek.com/) 获取）

### 安装步骤

1. **克隆或下载项目**
```bash
git clone <repository-url>
cd "Auto CV AI upload for jobsdb"
```

2. **安装依赖**
```bash
pip install -r requirements.txt
```

3. **安装 ChromeDriver**
   - 确保已安装 Chrome 浏览器
   - Selenium 4.x 会自动管理 ChromeDriver，通常无需手动安装
   - 如果遇到问题，可以从 [ChromeDriver](https://chromedriver.chromium.org/) 下载对应版本

## 使用说明

### 首次使用

1. **配置 API Key**
   - 打开程序，进入「1. 初始化配置」标签页
   - 输入你的 DeepSeek API Key
   - 点击「保存配置」

2. **配置浏览器（可选）**
   - 如果想使用已登录的 Chrome 账号，可以配置 Chrome 用户数据目录
   - 在 Chrome 地址栏输入 `chrome://version/` 查看「个人资料路径」
   - 将路径填入「Chrome用户数据目录」字段
   - 配置文件名称通常为 "Default" 或 "Profile 1"

3. **填写个人信息**
   - 填写姓名、邮箱、电话等信息
   - 这些信息将用于自动填写投递表单

### 使用流程

#### 方式一：手动单岗处理（推荐）

1. 进入「3. 手动单岗处理」标签页
2. 在「岗位链接」输入 JobsDB 岗位链接
3. 点击「抓取岗位信息」自动获取岗位描述
4. 在「原始简历」区域输入或上传你的简历（支持 Word 格式）
5. 选择简历语言（自动检测/中文/英文）
6. 点击「生成定制简历」
7. 查看生成的定制简历和匹配度
8. 可以导出为 PDF 或手动复制使用

#### 方式二：全自动求职

1. 进入「2. 全自动求职」标签页
2. 选择地区、输入搜索关键词和地点
3. 设置匹配度阈值（建议 70）
4. 点击「开始全自动求职」
5. 程序将自动搜索、生成简历并投递（需根据实际网页结构定制）

### 查看投递记录

1. 进入「4. 投递记录查询」标签页
2. 查看所有投递记录
3. 可以点击「刷新记录」更新列表
4. 点击「导出记录」保存为 CSV 或 Excel 文件

## 注意事项

1. **API 使用限制**：请注意 DeepSeek API 的使用限制和费用
2. **网站结构变化**：JobsDB 的网页结构可能变化，自动投递功能可能需要调整
3. **使用频率**：建议适当控制自动投递频率，避免被识别为机器人
4. **简历真实性**：生成的简历应保持真实性，建议人工审核后再投递
5. **隐私安全**：API Key 和配置文件包含敏感信息，请妥善保管

## 常见问题

### Q: 如何获取 DeepSeek API Key？
A: 访问 [DeepSeek Platform](https://platform.deepseek.com/)，注册账号并创建 API Key。

### Q: ChromeDriver 安装失败怎么办？
A: Selenium 4.x 会自动管理 ChromeDriver。如果仍有问题，确保 Chrome 浏览器已更新到最新版本。

### Q: Word 简历上传失败？
A: 确保已安装 `python-docx` 库：`pip install python-docx`

### Q: PDF 导出中文乱码？
A: PDF 导出需要中文字体支持。可以下载中文字体文件（如 simsun.ttc）并放在程序目录下。

### Q: 自动投递功能不工作？
A: 自动投递功能需要根据 JobsDB 的实际网页结构进行定制。建议先使用「手动单岗处理」功能。
