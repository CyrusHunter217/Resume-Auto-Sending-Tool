# JobsDB 自动求职工具 - EXE版本使用说明

## 重要提示

⚠️ **API Key 配置**：
- 程序**不会**在代码中包含任何API Key
- 首次运行时，必须在GUI界面中输入你的DeepSeek API Key
- API Key会保存在程序目录下的 `config.json` 文件中
- **不要**将包含API Key的 `config.json` 文件分享给他人

## 首次使用步骤

1. **运行程序**
   - 双击 `JobsDB自动求职工具.exe` 启动程序

2. **配置API Key**（必须）
   - 点击「1. 初始化配置」标签页
   - 在「DeepSeek API Key」输入框中输入你的API Key
   - 点击「保存配置」按钮
   - API Key会自动保存到 `config.json` 文件

3. **上传简历**
   - 在「3. 手动单岗处理」标签页
   - 点击「上传Word简历」按钮，选择你的简历文件
   - 或者直接在「原始简历」文本框中粘贴简历内容

4. **填写个人信息**（可选）
   - 在「1. 初始化配置」标签页
   - 填写姓名、邮箱、电话等信息
   - 点击「保存配置」

## 使用流程

### 方式一：全自动求职

1. 点击「2. 全自动求职」标签页
2. 选择地区、输入搜索关键词和地点
3. 设置匹配度阈值（建议70）
4. 点击「开始全自动求职」
5. 程序会自动搜索、筛选、生成简历

### 方式二：手动单岗处理

1. 点击「3. 手动单岗处理」标签页
2. 输入JobsDB岗位链接
3. 点击「抓取岗位信息」
4. 点击「生成定制简历」
5. 查看生成的简历，可以导出为PDF

## 配置文件说明

程序会在exe同目录下创建 `config.json` 文件，包含：
- API Key（加密存储，但建议不要分享）
- 简历内容
- 搜索条件
- 个人信息

**安全提示**：
- `config.json` 文件包含敏感信息
- 不要将此文件分享给他人
- 如果需要在其他电脑使用，请重新配置API Key

## 获取 DeepSeek API Key

1. 访问 https://platform.deepseek.com/
2. 注册/登录账号
3. 进入 API Keys 页面
4. 创建新的 API Key
5. 复制API Key并在程序中配置

## 常见问题

**Q: 程序提示"API Key未配置"？**
A: 请在「1. 初始化配置」标签页输入你的DeepSeek API Key并保存。

**Q: 如何清除所有配置？**
A: 点击「1. 初始化配置」标签页的「清除缓存」按钮，或直接删除 `config.json` 文件。

**Q: 可以在多台电脑使用吗？**
A: 可以，但需要在每台电脑上单独配置API Key。不要复制包含API Key的 `config.json` 文件。

**Q: 程序无法启动？**
A: 确保Windows系统已安装必要的运行库。如果仍有问题，请联系技术支持。
